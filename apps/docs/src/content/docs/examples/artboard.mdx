`@komeilm76/km-artboard` provides pure geometry functions for rectangular artboards. Every function is immutable — it returns new data rather than mutating inputs.

## Live example

Edit the code below and the output panel updates instantly.

<LiveExample
  client:only="react"
  title="Artboard quick start"
  code={`import { artboard } from '@komeilm76/km-geoboard';

// ── 1. Create an artboard ─────────────────────────────────────────────────
// createArtboard returns a discriminated union — check .success first
const r1 = artboard.createArtboard({
  startPoint: { x: 0, y: 0 },
  endPoint: { x: 800, y: 600 },
  name: 'Main canvas',
});
if (!r1.success) throw new Error(r1.reason);
const board = r1.artboard;
console.log('Created:', board.name, board.size.width + 'x' + board.size.height);

// ── 2. Move it ────────────────────────────────────────────────────────────
// moveArtboard is infallible — returns Artboard directly
const moved = artboard.moveArtboard({ artboard: board, delta: { x: 50, y: 50 } });
console.log('Moved origin:', moved.origin.x, moved.origin.y);

// ── 3. Resize it ──────────────────────────────────────────────────────────
const r3 = artboard.resizeArtboard({ artboard: moved, size: { width: 1200, height: 900 } });
if (!r3.success) throw new Error(r3.reason);
const resized = r3.artboard;
console.log('Resized to:', resized.size.width + 'x' + resized.size.height);

// ── 4. Snap to grid ───────────────────────────────────────────────────────
// snapArtboardToGrid is infallible — returns Artboard directly
const snapped = artboard.snapArtboardToGrid({ artboard: resized, gridSize: 100 });
console.log('Snapped origin:', snapped.origin.x, snapped.origin.y);

// ── 5. Point containment ──────────────────────────────────────────────────
const inside  = artboard.artboardContainsPoint(snapped, { x: 400, y: 300 });
const outside = artboard.artboardContainsPoint(snapped, { x: 5, y: 5 });
console.log('(400,300) inside:', inside);
console.log('(5,5) inside:', outside);

// ── 6. Overlap detection ─────────────────────────────────────────────────
const r6 = artboard.createArtboard({
  startPoint: { x: 700, y: 500 },
  endPoint:   { x: 1400, y: 1000 },
  name: 'Overlay',
});
const overlap = r6.success
  ? artboard.artboardsOverlap(snapped, r6.artboard)
  : false;
console.log('Boards overlap:', overlap);
`}
/>

## Key concepts

**Normalization** — `createArtboard` always produces a canonical artboard: `origin` is the top-left corner and `size` values are always positive. You can pass drag points in any order.

**Discriminated results** — fallible functions (`createArtboard`, `resizeArtboard`) return `{ success: true, artboard }` or `{ success: false, reason }`. Infallible functions like `moveArtboard` and `snapArtboardToGrid` return `Artboard` directly.

**Immutability** — no function mutates its input. Each call returns a new object.

## API summary

| Function | Returns | Notes |
|---|---|---|
| `createArtboard(input)` | `CreateArtboardResult` | Normalizes from two drag points |
| `resizeArtboard(input)` | `CreateArtboardResult` | Partial overrides for origin/size |
| `moveArtboard(input)` | `Artboard` | Translates by delta; infallible |
| `snapArtboardToGrid(input)` | `Artboard` | floor / round / ceil modes |
| `artboardContainsPoint(artboard, point)` | `boolean` | Inclusive bounds |
| `artboardsOverlap(a, b)` | `boolean` | True if any overlap exists |
| `artboardToRect / artboardFromRect` | `Rect / Artboard` | Convert to/from DOMRect-style |
