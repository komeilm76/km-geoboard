`@komeilm76/km-artboard` provides pure geometry functions for rectangular artboards. Every function is immutable — it returns new data rather than mutating inputs.

## Live example

Edit the code below and the output panel updates instantly.

<LiveExample
  client:only="react"
  title="Artboard quick start"
  code={`import {
  createArtboard,
  moveArtboard,
  resizeArtboard,
  snapArtboardToGrid,
  artboardContainsPoint,
  artboardsOverlap,
} from '@komeilm76/km-artboard';

// 1. Create from drag start/end points
const result = createArtboard({
  startPoint: { x: 100, y: 80 },
  endPoint:   { x: 400, y: 320 },
  name: 'Frame 1',
});

if (!result.success) {
  console.error('Failed:', result.reason);
  process.exit(1);
}

const { artboard } = result;
console.log('Created:', artboard.name, artboard.origin, artboard.size);

// 2. Move by a delta vector
const moved = moveArtboard({ artboard, delta: { x: 50, y: -10 } });
console.log('Moved origin:', moved.origin);

// 3. Partial resize (only width changes)
const resized = resizeArtboard({ artboard: moved, size: { width: 500 } });
if (resized.success) {
  // 4. Snap to 8 px grid
  const snapped = snapArtboardToGrid({ artboard: resized.artboard, gridSize: 8, mode: 'round' });
  console.log('Snapped size:', snapped.size);

  // 5. Point containment test
  const inside = artboardContainsPoint({ artboard: snapped, point: { x: 200, y: 200 } });
  console.log('Point inside?', inside);
}

// 6. Overlap detection
const other = createArtboard({ startPoint: { x: 350, y: 200 }, endPoint: { x: 600, y: 400 }, name: 'Frame 2' });
if (other.success) {
  const overlap = artboardsOverlap({ a: artboard, b: other.artboard });
  console.log('Overlaps?', overlap);
}
`}
/>

## Key concepts

**Normalization** — `createArtboard` always produces a canonical artboard: `origin` is the top-left corner, `size` is always positive. You can pass drag points in any order.

**Discriminated results** — fallible functions return `{ success: true, artboard }` or `{ success: false, reason }`. Infallible functions like `moveArtboard` return `Artboard` directly.

**Immutability** — no function mutates its input. Each call returns a new object.

## API summary

| Function | Returns | Notes |
|---|---|---|
| `createArtboard(input)` | `CreateArtboardResult` | Normalizes from two drag points |
| `resizeArtboard(input)` | `CreateArtboardResult` | Partial overrides for origin/size |
| `moveArtboard(input)` | `Artboard` | Translates by delta; infallible |
| `snapArtboardToGrid(input)` | `Artboard` | floor / round / ceil modes |
| `artboardContainsPoint(input)` | `boolean` | Inclusive bounds |
| `artboardsOverlap(input)` | `boolean` | True if any overlap exists |
| `artboardToRect / artboardFromRect` | `Rect / Artboard` | Convert to/from DOMRect-style |
