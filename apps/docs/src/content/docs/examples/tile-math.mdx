`@komeilm76/km-map` is a pure math library for web-map computations — no map renderer required. It covers Web Mercator tile indices, coordinate projections, great-circle distance, and bounding-box helpers.

## Live example

<LiveExample
  client:only="react"
  title="Tile math"
  code={`import { map } from '@komeilm76/km-geoboard';

// ── 1. Lat/lng → tile coordinates (zoom 10) ───────────────────────────────
const tile = map.lngLatToTile(13.405, 52.52, 10); // Berlin
console.log('Berlin tile z10:', tile);

// ── 2. Tile → centre lat/lng ──────────────────────────────────────────────
const centre = map.tileToLngLat(tile.x, tile.y, 10);
console.log('Tile centre:', centre);

// ── 3. Lat/lng → pixel within tile ───────────────────────────────────────
const px = map.lngLatToPixel(13.405, 52.52, 10);
console.log('Pixel (global):', px);

// ── 4. Haversine distance ─────────────────────────────────────────────────
const dist = map.haversineDistance(
  { lng: 13.405, lat: 52.52 },   // Berlin
  { lng: 2.349, lat: 48.864 },   // Paris
);
console.log('Berlin → Paris (km):', (dist / 1000).toFixed(1));

// ── 5. Tiles in bounding box ──────────────────────────────────────────────
const bbox = { west: 13.3, south: 52.4, east: 13.6, north: 52.6 };
const tiles = map.tilesInBounds(bbox, 12);
console.log('Tiles in Berlin bbox z12:', tiles.length);

// ── 6. Bounds containment ────────────────────────────────────────────────
const inside = map.boundsContains(bbox, { lng: 13.405, lat: 52.52 });
const outside = map.boundsContains(bbox, { lng: 2.349, lat: 48.864 });
console.log('Berlin in bbox:', inside, '| Paris in bbox:', outside);
`}
/>

## Coordinate systems

All functions work in **WGS84 / EPSG:4326** degrees. Tile XY follows the [OSM / Google Maps convention](https://wiki.openstreetmap.org/wiki/Slippy_map_tilenames) (origin top-left, Y increases downward).

## API summary

| Function | Returns | Notes |
|---|---|---|
| `lngLatToTile(lngLat, zoom)` | `TileXY` | Floor to integer tile indices |
| `tileToLngLat(tile, zoom)` | `LngLat` | Top-left corner of tile |
| `lngLatToPixel(lngLat, zoom)` | `Point` | Global pixel (256 px tiles) |
| `pixelToLngLat(pixel, zoom)` | `LngLat` | Inverse of above |
| `haversineDistance(a, b)` | `number` | Metres, WGS84 sphere |
| `tilesInBounds(bounds, zoom)` | `TileXY[]` | All tiles intersecting bbox |
| `boundsContains(bounds, point)` | `boolean` | Inclusive containment |
| `projectMercator(lngLat)` | `Point` | Web Mercator (EPSG:3857) metres |
| `unprojectMercator(point)` | `LngLat` | Inverse Mercator |
