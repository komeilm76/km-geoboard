`@komeilm76/km-svg` turns SVG XML strings into typed element trees, then converts path geometry into GeoJSON features — no browser, no canvas, no DOM required.

## Live example

<LiveExample
  client:only="react"
  title="SVG → GeoJSON"
  code={`import { svg } from '@komeilm76/km-geoboard';

// ── 1. Parse an SVG document ──────────────────────────────────────────────
const svgDoc = svg.parseSvgDocument('<svg viewBox="0 0 200 100"><rect x="10" y="10" width="80" height="60"/><circle cx="150" cy="50" r="30"/></svg>');
console.log('Elements found:', svgDoc.elements.length);
svgDoc.elements.forEach(el => console.log(' -', el.type, JSON.stringify(el.attributes)));

// ── 2. Convert individual SVG element to GeoJSON ──────────────────────────
const rectEl = svgDoc.elements[0];
const feature = svg.svgElementToGeoJSON(rectEl, { scale: 1 });
console.log('GeoJSON type:', feature.type, feature.geometry.type);

// ── 3. Convert whole SVG doc to GeoJSON FeatureCollection ─────────────────
const fc = svg.svgToGeoJSON(svgDoc, { scale: 1 });
console.log('FeatureCollection features:', fc.features.length);
fc.features.forEach((f, i) => console.log(' [' + i + ']', f.geometry.type));
`}
/>

## How it works

1. **`parseSvg`** lexes the SVG string into a `ParsedSvg` object containing a flat array of typed elements (`SvgRect`, `SvgCircle`, `SvgPath`, `SvgLine`, …). Every element carries its raw attributes and a normalized bounding box.

2. **`svgToGeoJSON`** maps each SVG element to the closest GeoJSON geometry:
   - `rect` → `Polygon` (four corners)
   - `circle` → `Polygon` (approximated with N-gon)
   - `path` → `LineString` or `MultiLineString` (M/L/Z commands parsed)
   - `line` → `LineString`

3. A **coordinate transform** option lets you provide a scale and origin to map SVG pixel space to geographic coordinates.

## API summary

| Symbol | Kind | Notes |
|---|---|---|
| `parseSvg(xml)` | Function | Returns `ParseSvgResult` |
| `svgToGeoJSON(elements, opts?)` | Function | Returns `SvgToGeoJSONResult` |
| `SvgElement` | Type union | `SvgRect \| SvgCircle \| SvgPath \| SvgLine \| SvgGroup \| SvgText` |
| `ParsedSvg` | Type | `{ elements, viewBox, width, height }` |
