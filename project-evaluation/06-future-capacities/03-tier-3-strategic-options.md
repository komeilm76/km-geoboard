# Tier 3 — Strategic options (choose at most one after v1.0)

1. **A hosted converter service** — the import/export matrix as a web API + UI
   (upload SVG/GeoJSON/KML → validated, converted, downloadable). Monetizable; modest ops.
2. **A Figma plugin product** — "export Figma frames as georeferenced SVG/GeoJSON";
   DT-Figma is half the spec already. Distribution via Figma Community.
3. **The geo-whiteboard** — the name "km-geoboard" implies it: artboards over basemaps,
   collaborative annotation, import/export everything. Largest build; only sensible after
   adapters + playground exist and have users.
