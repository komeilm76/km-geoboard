# Non-goals (guardrails — restate what DT-Logics forbids)
- No rendering, no DOM, no UI framework, no canvas/WebGL inside the packages.
- No map-engine dependency (MapLibre/Leaflet/OpenLayers are consumers, never dependencies).
- No exception-based APIs; no class hierarchies beyond the justified registry.
- No new feature work until G1 is met — stability before surface area.
